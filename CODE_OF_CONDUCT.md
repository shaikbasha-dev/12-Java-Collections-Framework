# Code of Conduct

## Our Commitment

We, as contributors and maintainers of this repository, are committed to making participation in our community a welcoming, respectful, and harassment-free experience for everyone, regardless of:

- Age
- Body size
- Disability
- Ethnicity
- Gender identity and expression
- Level of experience
- Nationality
- Personal appearance
- Race
- Religion
- Sexual identity and orientation
- Technical background

Our goal is to build a positive learning environment where everyone can collaborate, learn, and contribute with confidence and mutual respect.

---

# Our Standards

Examples of behavior that contribute to creating a positive environment include:

- Being respectful and considerate.
- Using welcoming and inclusive language.
- Accepting constructive feedback gracefully.
- Sharing knowledge to help others learn.
- Encouraging beginners and supporting fellow contributors.
- Maintaining professionalism in discussions.
- Respecting differing opinions and viewpoints.
- Reporting issues responsibly and respectfully.

---

# Unacceptable Behavior

The following behaviors are considered unacceptable:

- Harassment or discrimination of any kind.
- Offensive, abusive, or insulting language.
- Personal attacks or trolling.
- Intimidation or threats.
- Sharing another person's private information without permission.
- Posting spam or unrelated promotional content.
- Deliberately disrupting discussions or contributions.
- Any behavior that creates an unwelcoming environment.

---

# Scope

This Code of Conduct applies to all interactions related to this repository, including:

- GitHub Issues
- Pull Requests
- Discussions
- Documentation contributions
- Code contributions
- Community interactions related to this project

---

# Responsibilities

Repository maintainers are responsible for:

- Enforcing this Code of Conduct fairly and consistently.
- Reviewing reported incidents objectively.
- Taking appropriate action when necessary.
- Maintaining a safe and welcoming learning environment.
- Encouraging respectful collaboration.

---

# Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it by opening a private communication with the repository maintainer.

When reporting an issue, please include:

- A clear description of the incident.
- Relevant screenshots or evidence, if available.
- Links to the related discussion or Pull Request.
- Any additional context that may help in reviewing the report.

All reports will be reviewed respectfully and confidentially whenever possible.

---

# Enforcement

Project maintainers may take appropriate actions, including:

- Providing guidance or warnings.
- Requesting changes to inappropriate content.
- Removing offensive comments.
- Closing inappropriate issues or discussions.
- Restricting or blocking contributors who repeatedly violate this Code of Conduct.

Actions will be taken based on the severity and frequency of violations.

---

# Attribution

This Code of Conduct is inspired by the principles of the Contributor Covenant and adapted to support a professional, respectful, and educational open-source learning environment.
