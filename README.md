# 📖 About This Repository

The **Java Collections Framework (JCF)** is one of the most important topics in Java programming and is widely used in enterprise applications, backend development, Spring Boot projects, APIs, and technical interviews.

This repository has been designed as a **complete learning resource** that helps learners understand the Collections Framework from the fundamentals to advanced concepts in a structured and practical manner.

The repository covers:

- Collection Hierarchy
- List Implementations
- Queue Implementations

- Map Implementations
- Hashing Concepts
- Wrapper Classes
- Autoboxing and Unboxing
- Legacy Collection Classes
- Collections Utility Algorithms
- Internal Working of Collections
- Practical Java Programs
- Comparison Tables
- Frequently Asked Interview Questions

Every topic is organized into individual folders with dedicated documentation and Java programs to make learning simple, systematic, and interview-oriented.

# 📚 Repository Modules

## 📘 Module 01 – Introduction to Collections

Learn the fundamentals of the Java Collections Framework, understand why Collections were introduced, explore the limitations of arrays, study the advantages of Collections, understand the Collections hierarchy, and build a strong conceptual foundation.

---

## 📘 Module 02 – ArrayList

Learn one of the most widely used List implementations, understand dynamic array concepts, constructors, methods, internal working, CRUD operations, iteration techniques, comparisons, and interview questions.

---

## 📘 Module 03 – LinkedList

Understand doubly linked list implementation, insertion and deletion operations, internal working, advantages, limitations, traversal techniques, and practical Java programs.

---

## 📘 Module 04 – ArrayDeque

Study the implementation of double-ended queues, insertion and deletion from both ends, queue operations, iteration techniques, and interview-oriented concepts.

---

## 📘 Module 05 – PriorityQueue

Learn priority-based element storage, natural ordering, heap implementation, queue operations, internal behavior, and practical Java examples.

---

## 📘 Module 06 – HashSet

Understand unordered unique collections, hashing concepts, duplicate elimination, internal working, iteration methods, and commonly asked interview questions.

---

## 📘 Module 07 – LinkedHashSet

Learn insertion-order preservation, hashing with linked structure, practical examples, commonly used methods, and interview preparation.

---

## 📘 Module 08 – TreeSet

Master sorted collections, natural ordering, navigational methods, internal tree structure, practical programs, and interview concepts.

---

## 📘 Module 09 – Hashing

Gain an in-depth understanding of hashing mechanisms including hashCode(), equals(), buckets, collisions, load factor, hash tables, and rehashing techniques.

---

## 📘 Module 10 – HashMap

Learn key-value storage using hashing, insertion, retrieval, deletion, iteration, internal working, and performance characteristics.

---

## 📘 Module 11 – LinkedHashMap

Study ordered map implementation, insertion-order maintenance, methods, practical Java programs, and interview questions.

---

## 📘 Module 12 – TreeMap

Understand sorted maps, navigational methods, ordering concepts, internal tree implementation, and practical Java examples.

---

## 📘 Module 13 – Map Views

Learn how keySet(), values(), and entrySet() work, understand iteration techniques, and explore practical applications.

---

## 📘 Module 14 – Wrapper Classes

Understand primitive wrapper classes, their necessity, commonly used methods, comparisons, and practical Java examples.

---

## 📘 Module 15 – Autoboxing and Unboxing

Learn automatic conversion between primitive data types and wrapper objects along with practical examples and interview questions.

---

## 📘 Module 16 – Legacy Classes

Study Java's legacy collection classes including Vector, Stack, Hashtable, Dictionary, and Enumeration along with their practical usage.

---

## 📘 Module 17 – Collections Algorithms

Master the Collections utility methods including sorting, searching, reversing, shuffling, finding maximum and minimum elements, and practical implementations.

---

## 📘 Module 18 – Collection vs Collections

Understand the fundamental differences between the Collection interface and the Collections utility class using detailed explanations and comparison tables.

---

## 📘 Module 19 – Collections Hierarchy

Study the complete Java Collections Framework hierarchy including interfaces, implementation classes, inheritance relationships, and architecture.

---

## 📘 Module 20 – Collections Interview Questions

Revise beginner, intermediate, advanced, scenario-based, and frequently asked MNC interview questions to strengthen interview preparation.
